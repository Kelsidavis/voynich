### 3.3 Medical Instruction Patterns

Pattern 1: **Patient + Ingredient + Instruction**
```
"daiin...chor...dar" = "leaf...SICK...GIVE"
→ "Give leaf [preparation] to the sick"
```

Pattern 2: **Body Part + Preparation**
```
"oky...ol...dal" = "EYES...OIL...GAVE"
→ "[For] eyes, oil, [was] given"
```

### 3.4 Interpretation
The herbal section functions as a **materia medica index** where each plant illustration is accompanied by:
- Plant identification (Latin botanical terms)
- Medical indication (Polish patient references)
- Administration method (Polish instruction verbs)
