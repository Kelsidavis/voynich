# Voynich Biological Section Nymph Labels - Decoding Attempt

## Section Overview

The biological section (f75-f84) contains drawings of female figures ("nymphs") often depicted in pools or connected by tubes/pipes. Each figure typically has an associated label.

### Label Types in Transcription

| Tag | Meaning | Description |
|-----|---------|-------------|
| @Ln | Label, nymph | Primary nymph label |
| =Ln | Secondary nymph label | Additional nymph label |
| ~Ln | Uncertain nymph label | Less clear label |
| @Lt | Label, tube | Pipe/tube label |
| =Lt | Secondary tube label | Additional tube label |
| @P0 | Paragraph | Main text near figures |
