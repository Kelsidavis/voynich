# Basque Verbs and Action Words in Voynich Manuscript

## Test Purpose

If Voynich is related to Basque, we might find Basque verb roots in:
- Recipe sections (action instructions)
- Herbal sections (preparation methods)
- Any procedural text

Note: We expect verb ROOTS, not conjugated forms, since Voynich grammar differs from Basque.
