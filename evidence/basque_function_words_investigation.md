# Basque Function Words and Grammar Particles in Voynich Manuscript

## Test Purpose

Function words (conjunctions, pronouns, articles, case markers, postpositions) are among the most stable and frequent elements in any language. If Voynich is related to Basque, we might find:
- Basque conjunctions (eta, edo, baina)
- Basque pronouns (ni, zu, hura)
- Basque case suffixes (-an, -eko, -ra)
- Basque postpositions (gabe, arte, ondoren)

### Key Insight

**The ABSENCE of Basque function words SUPPORTS rather than refutes the sister language hypothesis.**

If Voynich were simply "Basque in a different script," we would expect:
- eta (and), edo (or) → ABSENT
- ni, zu, hura (pronouns) → ABSENT
- -ak, -ari, -rekin (case suffixes) → ABSENT

- Shared content vocabulary (plants, body, verbs)
- Completely different grammatical system

**This is exactly what a pre-historic language split would produce.**
