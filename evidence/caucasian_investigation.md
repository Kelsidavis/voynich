# Investigation: Caucasian Languages Connection to Voynich

## Background

The Caucasus region contains multiple language families:

1. **Kartvelian** (South Caucasian)
   - Georgian, Mingrelian, Svan, Laz
   - Agglutinative with complex verb morphology

2. **Northeast Caucasian** (Nakh-Dagestanian)
   - Chechen, Ingush, Avar, Lezgin
   - Complex consonant clusters, ergative alignment

3. **Northwest Caucasian** (Abkhaz-Adyghe)
   - Abkhaz, Circassian (Adyghe, Kabardian)
   - Extremely complex consonant systems

4. **Armenian** (Indo-European, but geographically Caucasian)
   - Unique branch of Indo-European
   - Rich morphology

The Caucasus was on trade routes connecting Europe to Asia, making it relevant to test.

## Test 4: Caucasian Phonological Features

### Consonant Clusters

Caucasian languages are famous for complex consonant clusters:
- Georgian: მწვერვალი (mts'vrvali) - 5 consonants in a row
- Abkhaz: Can have 4+ consonants together
- Circassian: Extremely consonant-heavy

### Voynich Consonant Patterns

| Pattern | Examples | Count |
|---------|----------|-------|
| ch- | chedy, chol, chor | 1696 |
| sh- | shedy, shol, shor | 1072 |
| qo- | qokaiin, qokedy | 4991 |
| lch- | lchedy | 119 |

Voynich has some clusters (ch, sh, qo, lch) but NOTHING like Caucasian complexity.

**Result: Voynich phonology does NOT match Caucasian patterns.**

## Test 5: Grammatical Structure

### Georgian Verb Structure

Georgian verbs can have:
- Multiple prefixes (up to 4)
- Multiple suffixes
- Subject AND object markers

Example: გავაკეთებინებდით (gavak'etebinebdit)
= "we would have had someone make it"

Voynich structure is MUCH simpler:
- One prefix (qo-, o-, y-)
- One suffix (-aiin, -dy, -ol, etc.)

**Result: Voynich grammar is NOT Caucasian-complex.**

### Why Caucasian Languages Don't Match

1. **Phonological mismatch**: Voynich lacks complex consonant clusters
2. **Grammatical simplicity**: Voynich structure much simpler than Caucasian
3. **Vocabulary gaps**: Basic words (water, sun, moon) don't match
4. **Case suffixes absent**: No Georgian -i/-ma/-s or Armenian markers
